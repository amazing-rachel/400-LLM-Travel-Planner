# LLM Travel Planner

## Introduction
The LLM Travel Planner is a web application designed to simplify trip planning by generating personalized, day-by-day itineraries. Using a Large Language Model (LLM) and the user's
own personal preferences, the web app provides a list of suggested itineraries for a more personalized trip. The user can save, rank and remove from the list of itineraries for future
use as well.

## Technical Stack
- Frontend Technologies: HTML, CSS, JavaScript, and React
- Backend Language: Python
- Database & AI Database: SQL (SQLite) Database
- AI Model: Gemini 3 Flash
